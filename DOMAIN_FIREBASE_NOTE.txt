Zest OTT Firebase Login Notes

Cloudflare domain:
Yes, Google login can work on zestotts.shop when hosted through Cloudflare.

Required Firebase settings:
1. Firebase Authentication > Sign-in method > Google must be enabled.
2. Firebase Authentication > Settings > Authorized domains must include:
   - zestotts.shop
   - www.zestotts.shop
   - localhost only if testing locally

Important:
If Cloudflare is only managing DNS, login still works.
If using Cloudflare Pages, add the Cloudflare Pages domain too:
example.pages.dev

Track order:
For a real track-order system, use Firebase Firestore.
You need:
- orders collection
- orderId
- customerEmail
- product
- plan
- paymentMethod
- receiptStatus
- orderStatus
- createdAt

Admin management:
You can manage order status either:
1. Directly from Firebase Console > Firestore
2. Or with a custom admin.html page later
